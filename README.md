# 💊 MediVault - Smart Medicine Reminder System

MediVault is a smart medicine management application built using Flutter that helps users manage their medications, set reminders, track medicine history, and improve medication adherence.

The application provides a simple and user-friendly interface to store medicine details, receive timely reminders, and maintain personal health records digitally.

---

## 🚀 Features

### 💊 Medicine Management
- Add, edit, and delete medicines
- Store medicine name, dosage, schedule, and instructions
- View all active medications in one place

### ⏰ Smart Medicine Reminders
- Set reminders for medication timings
- Get notifications when it is time to take medicine
- Reduce missed doses with automated alerts

### 📅 Medication Schedule
- Daily medicine timetable
- Track upcoming doses
- View medicine history

### 📊 Health Tracking
- Monitor medicine intake patterns
- Maintain digital medication records
- Improve medication consistency

### 🔐 Secure Storage
- Store user medicine data safely
- User-friendly and privacy-focused design
